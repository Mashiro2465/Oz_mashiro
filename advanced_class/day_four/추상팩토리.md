# 추상 팩토리(Abstract Factory) 패턴

## 정의 
서로 연관되거나 의존적인 객체들의 **구체적인 클래스 지정 없이** 객체들을 생성할 수 있도록 하는 생성 패턴

## 목적
- 관련된 객체들을 **일관성 있게 생성**
- 구체 클래스에 대한 의존성 제거
- 제품군을 쉽게 교체 가능

## 구조

### 구성 요소

### AbstractProduct (추상 제품)
- 제품 인터페이스 정의

### AbstractFactory (추상 팩토리)
- 제품 생성 메서드를 선언

### ConcreteProduct (구체 제품)
- 실제 생성되는 객체

### ConcreteFactory (구체 팩토리)
- 실제 객체 생성 구현
