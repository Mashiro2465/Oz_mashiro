# 팩토리 메서드

## 1. Simple Factory (간단한 팩토리)

- 조건에 따라 객체를 만들어주는 함수 or 클래스

### 언재 쓰는지
- 입력값에 따라 객체를 골라야 할 때
- 구조를 단순하게 유지하고 싶을 때

### 특징
- 디자인 패턴은 아님


## 2. Factory Method (팩토리 메서드)

- 객체 생성은 서브클래스에게 맡기고, 사용 흐름은 상위 클래스에서 고정

### 언재 쓰는지
- 전체 흐름, 알고리즘은 같음
- 객체 생성의 단계만 다를때

### 특징
- 디자인 패턴
- 확장에 강함


| | 간단한 팩토리 | 팩토리 메서드 |
|-|---------|---------|
| 객체 선택 방식 | if      | 상속      |
| 생성 책임 | 한 클래스   | 서브클래스   |
| GoF 패턴 | X       | O       |
| 실무 사용 | 매우 많음   | 특정 상황   |
